# realtimeBingoApp

リモートでビンゴを遊ぶことを想定。ルームを作成し、参加することでリアルタイムでビンゴ抽選を実現

フロント：React.js

バック:Node.js

DB:Mysql

ルーム作成はWebSocketを使用

cd bingo-app-sever →　node app.jsでローカル起動
